# 🌐 Perfil Pessoal

Página de perfil pessoal desenvolvida como primeiro projeto prático.

## 🚀 Como visualizar
Acesse: clique aqui (https://marcelosalatino-crypto.github.io/Perfil-Pessoal)

## 🛠️ Tecnologias utilizadas
- HTML5 — estrutura da página
- CSS3 — estilização e dark mode
- JavaScript — efeito de digitação animado

## ✨ Funcionalidades
- Tag animada "Disponível para estágio"
- Efeito de digitação no cargo
- Cards de tecnologias com hover
- Links para GitHub, LinkedIn e Email
- Layout responsivo para celular
